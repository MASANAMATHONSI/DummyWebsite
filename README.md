African Flowers Interactive Floral Learning
Flowerology is an educational web application designed to help users explore the fascinating world of botany. Built with a focus on interactivity, 
the site provides detailed biological insights into various flowers through a modern, responsive interface.

 Features
1. Interactive Homepage
About Section: A central hub providing the mission and vision of the platform.
Feature Toggles: Dynamic toggles that reveal deep-dives into flower species.
Biology Deep-Dive: A unique UI transition where the homepage content is replaced with comprehensive biological data across 60+ different categories.
Hover Interactivity: Sophisticated CSS "Blur & Reveal" effects. When hovering over specific flowers (like sunflowers), the background blurs to prioritize a pop-up containing detailed information.

3. Educational Quiz Engine
Dynamic Questioning: Powered by JavaScript to test user knowledge on botany.
Real-time Scoring: Tracks user performance throughout the session.
Performance Feedback: Provides a final score summary based on the answers provided.

5. Feedback System
Contact Us: A dedicated form for users to provide feedback or inquiries, ensuring a two-way communication channel between the user and the developer.

   Tech Stack
Frontend: HTML5, CSS3
Logic: JavaScript (ES6+)
Styling: Custom CSS Transitions & Filter Effects (Blur)
